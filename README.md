
# Online Travel Booking System

## Features
• User Register & Login  
• Travel Package Booking  
• Booking History  
• Cancel Booking  
• Data Validation  
• Error Handling  
• Secure Password Hashing  

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Node.js, Express  
Database: MongoDB  

## Setup Instructions
1️⃣ Open terminal  
2️⃣ Go to backend folder  
3️⃣ Run following commands  

npm install  
node server.js  

Ensure MongoDB is running.
